## Tech stacks

Tecnologías y plataformas con las que trabajo en los proyectos:

**Cloud y arquitectura:** AWS (Well-Architected, Bedrock, Bedrock AgentCore, AgentCore Gateway, Knowledge Base, Neptune), Azure, Oracle Cloud (OCI, OKE), Cloudflare, on-premises, híbrido, Terraform, arquitectura de sistemas, optimización de coste y rendimiento

**IA y ML:** IA generativa, LLMs, RAG, flujos multi-agente, sistemas agénticos, LlamaIndex, OpenAI, Azure Cognitive Services, Azure AI Foundry, LocalAI, embeddings y búsqueda vectorial

**Ingeniería:** Go, Python, C, C++, TypeScript, JavaScript, Node.js, Svelte, HTML, CSS, Flutter

**Plataforma y ops:** Kubernetes, Docker, Docker Compose, Traefik, Modal, Argo Workflows, Prometheus, OpenTelemetry

**Datos y mensajería:** MongoDB, CosmosDB, Redis, ServiceBus, EPUB

**DevOps y herramientas:** Git, GitHub, Bitbucket, Azure DevOps, Jenkins, Azure Pipelines, GitHub Actions, CI/CD

**Prácticas:** Clean Code, TDD, SOLID, Kanban, SCRUM, Agile, 8D, transferencia de conocimiento
