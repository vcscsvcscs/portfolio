## Tech stackek

Technológiák és platformok, amelyekkel megbízások során dolgozom:

**Felhő & architektúra:** AWS (Well-Architected, Bedrock, Bedrock AgentCore, AgentCore Gateway, Knowledge Base, Neptune), Azure, Oracle Cloud (OCI, OKE), Cloudflare, on-premises, hibrid, Terraform, rendszerarchitektúra, költség- és teljesítményoptimalizáció

**AI & ML:** Generatív AI, LLM-ek, RAG, multi-agent workflow-k, ágens alapú rendszerek, LlamaIndex, OpenAI, Azure Cognitive Services, Azure AI Foundry, LocalAI, embedding és vektoros keresés

**Fejlesztés:** Go, Python, C, C++, TypeScript, JavaScript, Node.js, Svelte, HTML, CSS, Flutter

**Platform & ops:** Kubernetes, Docker, Docker Compose, Traefik, Modal, Argo Workflows, Prometheus, OpenTelemetry

**Adat & üzenetkezelés:** MongoDB, CosmosDB, Redis, ServiceBus, EPUB

**DevOps & eszközök:** Git, GitHub, Bitbucket, Azure DevOps, Jenkins, Azure Pipelines, GitHub Actions, CI/CD

**Gyakorlatok:** Clean Code, TDD, SOLID, Kanban, SCRUM, Agilis, 8D, tudásmegosztás
