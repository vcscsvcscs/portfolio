## Code Budapest #LikeABosch 2022
**2022**  
**Budapest, Hungría**

Participé en el Code Budapest #LikeABosch 2022 hackathon con mi equipo, donde trabajamos en el Desafío de Software. Este evento fue un hito significativo en mi carrera, ya que condujo directamente a mi reclutamiento por Bosch.

![Code Budapest #LikeABosch 2022 Hackathon](/images/code-like-a-bosch.png)

## El Desafío

Abordamos el desafío de procesar datos del array de sensores de Ego para construir un sistema integral de seguimiento de objetos alrededor de vehículos. La innovación clave fue nuestra capacidad de rastrear objetos incluso cuando estaban en puntos ciegos de sensores, abordando un desafío crítico en sistemas de percepción de vehículos autónomos.

## Nuestra Solución

Nuestro enfoque se centró en:
- **Procesamiento de Datos de Sensores**: Analizar y procesar datos del array multi-sensor de Ego
- **Seguimiento de Objetos**: Construir un sistema de seguimiento robusto que mantiene la identidad del objeto a través de áreas de cobertura de sensores
- **Manejo de Puntos Ciegos**: Implementar algoritmos para predecir y rastrear objetos incluso cuando se mueven a través de áreas no cubiertas directamente por sensores
- **Rendimiento en Tiempo Real**: Asegurar que el sistema pueda procesar datos de sensores en tiempo real

## Stack Tecnológico

Construimos nuestra solución usando:
- **Python 3**: Lenguaje de desarrollo principal
- **Python Arcade**: Para visualización y demostración del sistema de seguimiento
