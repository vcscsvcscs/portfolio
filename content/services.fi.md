## Tech stackit

Teknologiat ja alustat, joita käytän projekteissa:

**Pilvi ja arkkitehtuuri:** AWS (Well-Architected, Bedrock, Bedrock AgentCore, AgentCore Gateway, Knowledge Base, Neptune), Azure, Oracle Cloud (OCI, OKE), Cloudflare, on-premises, hybridi, Terraform, järjestelmäarkkitehtuuri, kustannus- ja suorituskykyoptimointi

**Tekoäly ja ML:** Generatiivinen tekoäly, LLM:t, RAG, moniagenttiset työnkulut, agenttiset järjestelmät, LlamaIndex, OpenAI, Azure Cognitive Services, Azure AI Foundry, LocalAI, upotukset ja vektorihaku

**Kehitys:** Go, Python, C, C++, TypeScript, JavaScript, Node.js, Svelte, HTML, CSS, Flutter

**Alusta ja ops:** Kubernetes, Docker, Docker Compose, Traefik, Modal, Argo Workflows, Prometheus, OpenTelemetry

**Data ja viestintä:** MongoDB, CosmosDB, Redis, ServiceBus, EPUB

**DevOps ja työkalut:** Git, GitHub, Bitbucket, Azure DevOps, Jenkins, Azure Pipelines, GitHub Actions, CI/CD

**Käytännöt:** Clean Code, TDD, SOLID, Kanban, SCRUM, Agile, 8D, tiedonsiirto
